## Description

Akasha is aim to be the backend service for tracking the crypto potfolio. The service will be used to keep the track of all your staking , defi activity and also the the deposited tracking of the FIAT asset into crypto.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
